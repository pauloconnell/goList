Exercise Tracker


Simple app to create a user in the database with a unique userID and store exercises. Back End API set up to be queried with parameters as described at bottom of page.
Successful Tracking of your exercise!

Add exercises

Use your unique userID to log your exercises


ADVANCED USERS: API Query syntax for exercise log:
GET /api/exercise/log?{userId}[&from][&to][&limit]

{ } = required, [ ] = optional from, to = dates (yyyy-mm-dd); limit = number
