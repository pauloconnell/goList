Exercise Tracker

Back End: API set up to be queried with parameters as described at bottom of page.

Get a list of your exercise logs:

To get an Array of users get: /api/exercise/users/ OR CLICK HERE

ADVANCED USERS: API Query syntax for exercise log:
GET /api/exercise/log?{userId}[&from][&to][&limit] { } = required, [ ] = optional from, to = dates (yyyy-mm-dd); limit = number
